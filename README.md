# Personal Blog

April 2023 I wanted a new blog! I archived the [old](https://github.com/mile95/old-blog) blog but stole the "best" posts and created this new blog instead.

This one runs using `hugo` instead of `jekyll`, thanks to [gokarna](https://github.com/526avijitgupta/gokarna) for the nice theme.

## Running locally

    hugo serve
